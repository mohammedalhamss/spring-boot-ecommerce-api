## 📖 Project Overview
[cite_start]The Online Shop Management System is a comprehensive RESTful web application developed using Spring Boot[cite: 12]. It provides a complete e-commerce experience, featuring a dynamic frontend interface connected to a robust backend infrastructure. [cite_start]The system supports distinct roles for Users (shopping, cart management, checkout) and Administrators (inventory, user, and payment management)[cite: 206, 229, 230].

## ✨ Key Features
**User Features:**
* [cite_start]**Customer Registration & Authentication:** Secure session-based login[cite: 14, 226].
* [cite_start]**Shopping Dashboard:** Browse available products and view details dynamically[cite: 301, 308].
* [cite_start]**Cart Management:** Add products, adjust quantities, and view total prices[cite: 309].
* [cite_start]**Checkout & Payments:** Process payments securely and view individual payment history[cite: 16, 17, 311].

**Admin Features:**
* [cite_start]**Product Management:** Add, update, and monitor product stock[cite: 15, 253].
* [cite_start]**User Management:** View and manage registered platform users[cite: 253].
* [cite_start]**Financial Oversight:** Access detailed payment statements, sales history, and transaction statuses[cite: 253].

## 🛠️ Technologies Used
* [cite_start]**Backend:** Java, Spring Boot, Spring Data JPA[cite: 12, 315].
* [cite_start]**Frontend:** HTML, CSS, JavaScript (Vanilla JS for DOM manipulation and Fetch API for backend communication)[cite: 314].
* [cite_start]**Architecture:** Layered Architecture (Controller, Service, Repository, Entity)[cite: 18].
* [cite_start]**Data Format:** JSON for RESTful API communication[cite: 323].
